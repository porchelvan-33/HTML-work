Common HTML Tags and Descriptions

Tag Description Example
<!DOCTYPE html> Defines the document type and HTML version. <!DOCTYPE html>
<html> The root element of an HTML page. <html>...</html>
<head> Contains meta-information about the HTML page. <head>...</head>
<title> Specifies a title for the HTML page (which is shown in the browser's title bar or in the page's tab). <title>My Page</title>
<body> Defines the document's body. <body>...</body>
<h1> to <h6> Defines HTML headings. <h1>Heading 1</h1>
<p> Defines a paragraph. <p>This is a paragraph.</p>
<a> Defines a hyperlink. <a href="https://www.example.com">Link</a>
<img> Defines an image. <img src="image.jpg" alt="My Image">
<ul> Defines an unordered list. <ul><li>Item 1</li><li>Item 2</li></ul>
<ol> Defines an ordered list. <ol><li>Item 1</li><li>Item 2</li></ol>
<li> Defines a list item. <li>List Item</li>
<div> Defines a section in a document. <div>...</div>
<span> Defines an inline section in a document. <span>Text</span>
<br> Inserts a single line break. Line 1<br>Line 2
<hr> Defines a horizontal rule. <hr>
<table> Defines a table. <table>...</table>
<tr> Defines a table row. <tr>...</tr>
<th> Defines a table header. <th>Header</th>
<td> Defines a table cell. <td>Data</td>
<form> Defines an HTML form for user input. <form>...</form>
<input> Defines an input control. <input type="text">
<label> Defines a label for an input element. <label for="username">Username:</label>
<button> Defines a clickable button. <button>Click Me</button>
<select> Defines a dropdown list. <select>...</select>
<option> Defines an option in a dropdown list. <option value="value">Option</option>
<textarea> Defines a multiline input control. <textarea>...</textarea>
<script> Defines a client-side script. <script>...</script>
<style> Defines style information for a document. <style>...</style>
<meta> Defines metadata about an HTML document. <meta charset="UTF-8">
